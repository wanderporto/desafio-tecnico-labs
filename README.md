# Tracking Api

Api responsavel para agendamento e envio de comunicação


## Requisitos
```sh
Maven
Java 11
Docker Compose
```

## Iniciando
```
clonar repositório
- `git clone https://github.com/wanderporto/desafio-tecnico-labs.git`

entrar na pasta do projeto
cd exemplo

executar o projeto
./mvnw spring-boot:run
```

## Stack de tecnologias utilizadas
```
Java 11
Spring Boot
Jpa/Hibernate
Postgresql
Junit
swagger
```

## end-points
com o Swagger: 
http://localhost:8080/swagger-ui.html#/
